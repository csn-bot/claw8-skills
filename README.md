# claw8 skills

This repo contains reusable "skills" and helpers used in the `claw8` workspace.

## Contents
- `dayone-crud/`: Create, read, and update Day One journal entries via MCP (and optional SQLite text reads).
- `markdown2slides/`: Slide-ready Markdown or PPTX → Moodle-paste HTML (`scripts/markdown2slides.py`).
- `ppt2markdown/`: PPTX → slide-ready Markdown → Day One (`scripts/ppt2markdown.py`).
